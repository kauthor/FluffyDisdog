[레벨·맵 디자인용] 스테이지 구조 및 보상 흐름 요약
출처: 땅부수기.pptx, 땅부수기2026_ch2.pptx
========================================

■ 스테이지 유형
- 일반 스테이지: 기본 타일 파괴 목표 달성
- 보스 스테이지: 3스테이지마다 등장, 특수 기믹 포함
- 총 21스테이지 (7묶음 × 3스테이지) 정상 진행

■ 보스 스테이지 기믹 예시
- 안개 기믹: 방해 타일 위치 비공개
- 전염 기믹: 방해 타일이 주변 일반 타일을 방해 타일로 전염
- 불안정 기믹: 방해 타일 타격 시 불안정 게이지 +1, 10 달성 시 게임 오버

■ 타일 구성 요소
- 일반 타일: 기본 파괴 대상
- 방해 타일: 파괴 금지 (일부 카드/유물로 파괴 가능)
- 도움 타일: 파괴 시 추가 보상
- 보물 타일: 덮고 있는 타일 전부 파괴 시 보상 획득
- 타일 상태: 균열 / 전염 / 암흑 / 잔상

■ 난이도 상승 구조
- 매 스테이지: 목표 점수 상승 (y = x²/4 올림)
- 3스테이지마다: 목표 점수 추가 상승 (y' = z³)
- 5스테이지마다 방해 타일 수 증가 / 도움 타일 수 감소
- 5스테이지 클리어마다 보상: 도움 타일 증가 / 손패 확장 등

■ 스테이지 클리어 보상 흐름
1. 카드 보상 선택 (1~3장 중 선택)
2. 보물 보상 수령 (해당 스테이지에서 획득한 경우)
3. 상점 진입

■ 카드 보상 구성
- 기본: 일반 카드 위주
- 확률적으로 등장: 카드 제거, 카드 강화 카드, 골드 카드
- 후반 스테이지: 고밸류 카드 / 카드 제거 / 강화 카드 확률 증가

■ 보물 시스템 상세
- 생성 위치: 랜덤, 파괴되지 않은 타일 아래 숨겨짐
- 기본 생성 수: 스테이지당 1개 (유물로 증가 가능)
- 등급별 크기:
  - 초급: 2x1, 1x2, 2x2 (3종)
  - 중급: 2x2 (1종)
  - 상급: 2x2, 2x3, 3x2, 1x1 (4종)
- 등급별 확률: 상급 5% / 중급 20% / 초급 나머지
- 보상 선택지: 5개 제공, 골드 1개 보장
  - 초급: 골드(소), 일반 카드
  - 중급: 골드(소~중), 일반~레어 카드, 유물
  - 상급: 골드(소~대), 모든 카드, 모든 유물

■ 보상 획득창 UI 구조 (땅부수기2026_ch2.pptx)
- 게임 도중 등장 시 접기/펼치기 버튼 제공
- Select all 버튼, 선택 수량 표기 (n/m)
- 클릭 한 번으로 선택, 우하단 Next 버튼으로 진행
- 아무것도 선택 안 하면 Skip으로 표기

■ 상점 진입 후 구조
- A구역: 낱장 카드 2장
- B구역: 유물/포션 랜덤 3개
- C구역: 팩 4개
- 의뢰 슬롯: 항상 1개 고정
- 리롤: 낱장 카드 제외 전체 리롤 가능
