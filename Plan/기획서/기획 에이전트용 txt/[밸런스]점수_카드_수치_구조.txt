[시스템 밸런스용] 점수 및 카드 수치 구조 요약
출처: 땅부수기.pptx, 카드_정리.pptx
========================================

■ 점수 구조
- 기본 파괴 점수: 타일 1개 파괴 시 기본 점수 획득
- 태그 점수: 카드 태그(도구/황금 등)에 따라 추가 점수 적용
- 타일 상태 점수: 균열/잔상/연소/빙결/감전 등 타일 상태별 별도 점수 계산
- 점수 배율: 유물/태그 조합으로 배율 스택 가능

■ 골드 구조
- 시작 골드: 100G
- 목표 초과 파괴: 초과분 골드 환산 지급
- 전체 타일 파괴 보너스: 추가 골드 지급
- 상점 리롤 비용: 골드 소모
- 의뢰 투자: 5G(일반) / 15G(희귀) / 30G(유니크)

■ 난이도 곡선 (게임_종료_조건.txt 기준)
- 1스테이지 요구: 8점
- 매 스테이지: y = (x²)/4 올림값 추가
- 3스테이지마다: y' = z³ 올림값 추가
- 22스테이지 요구: 1220점 (사실상 종료)

■ 카드 등급 구조
- 일반(Rarity 1): 기본 카드, 도구 태그 위주
- 레어(Rarity 2): 강화 카드, 특수 효과 보유
- 유니크(Rarity 3): 고스펙 특수 카드

■ 도구 태그 카드 파괴 확률 구조
- 기본: 0% 파괴 확률
- 카드 사용 시마다 5%씩 증가
- 최대 70% (삽/황금삽 기준), 최대 50% (횃불/기억의 망치 기준)
- 강화(+) 카드: 파괴 효과 제거

■ 카드별 타격 확률 구조
- 삽: 1칸 100% 파괴
- 갈퀴/도끼: 중심 100% + 주변 66%
- 곡괭이/나무망치: 중심 100% + 주변 50%
- 대형가위: 중심 100% + 주변 75%
- 새총: 랜덤 5칸 33% 파괴
- 모서리 파쇄 해머: 중심 100% + 상하좌우 66% + 대각선 33%
- 집중형 십자드릴: 중심 100% + 대각선 66% + 상하좌우 33%

■ 소모형 카드 구조
- 가방: 덱 최상단 1장 드로우
- 도구 변형: 손패 도구 카드 → 덱 내 도구 카드로 교체
- 도구 강화: 다음 턴 도구 카드 점수 2배

■ 황금 태그 카드
- 황금 삽/황금 도끼: 사용 시 1골드 추가 획득
- 균열 지층 분석: 균열 타일 파괴 시 2G 지급, 5칸 파괴 시 황금 카드 드로우

■ 의뢰 시스템 수치
- 성공 확률: 기본 50%, 매 상점 방문 +10%, 투자 시 +10%
- 대성공 확률: 성공 시 별도 체크, 기본 30%, 투자 시 +10%
- 투자 횟수: 최대 3회
