의뢰 기획서

//  ★: 구현할 때 확인해야 하는 부분

의뢰
 - 상점에서 재화를 투자하고, 다음 상점 방문시 최소 그 재화가치 혹은 그 이상의 보상으로 돌려받을 수 있는 기능

의뢰 방식
 - 상점에 입장시 상점 내부에서 의뢰할 수 있음 ★
 - 동일한 상점 방문 이미 의뢰가 진행중이거나, 보상을 받았다면 의뢰할 수 없음 ★
 - 의뢰는 각 5G, 15G, 30G가 들어가는 일반 의뢰, 희귀 의뢰, 유니크 의뢰가 있어 하나를 선택할 수 있다.

 // 아직 재화의 가치가 정해지지 않았으므로 일단 고정 값으로 구현하고 테이블 화가 필요해보임.

투자 방식
 - 상점에 입장시 이미 의뢰가 진행중이고, 보상을 받지 않았다면 투자를 할 수 있음  ★
 - 동일한 상점 방문에서 의뢰를 했다면 해당 방문에서는 투자를 할 수 없음 // 의뢰 슬롯은 항상 하나로 고정
 - 선택한 의뢰와 동일한 금액의 의뢰비를 추가로 지불한다 ★
 - 이후 보상에서 성공률과 대성공 확률이 상승한다.  ★

수령 방식
 - 상점에 입장시 이미 의뢰가 진행중이고, 투자를 진행하지 않았다면 보상을 수령할 수 있음 // 투자 진행 or 넘어가기
 - 성공 확률은 기본 50%, 매 상점 방문시 10% 증가, 투자시 10% 증가
 - 대성공 확률은 성공시 별개로 체크. 기본 30%, 투자시 10% 증가

 // 확률의 경우 테이블 화 필요.

    
    
//보상의 경우 확률에 따라 3가지로 구분

보상(실패시) ★
 - 한번의 보상 선택창이 등장함 
 - 투자한 등급과 동일한(일반,레어,유니크) 카드 1장, 전체랜덤 유물 1개, 전체랜덤 소모품 1개, 카드제거/강화 중 1개 중에서 1택

 // 보상 테이블 참조

보상(성공시) ★
 - 한번의 보상 선택창이 등장함. (카드팩 선택시 카드팩 구매와 동일한 창으로 넘어감)
 - 전체랜덤 카드팩 1종(등급 보장), 덱에 들어있는 카드 태그와 관련된 유물 1개, 카드제거/강화 중 1개 중에서 1택

 // 보상 테이블 참조

보상(대성공시) ★
 - 보상 선택창 없이 모두 지급함. (원한다면 미수령 가능)
 - 덱에 들어있는 카드 태그와 관련된 카드팩 1개(등급 보장)
 - 덱에 들어있는 카드 태그와 관련된 유물 1개
 - 카드제거/강화 중 1장

 // 보상 테이블 참조
