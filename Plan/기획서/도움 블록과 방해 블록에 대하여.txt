도움 블록과 방해 블록에 대하여.

방해 블록 (출처: 땅부수기.pptx 4p)
    1. 포인트 요구치가 상승하며 추가적으로 일반 타일이 생성됨 
        (몇 개 생성되는지/일반 타일이 늘어나는 점이 왜 패널티인지 설명 필요/포인트 요구치는 얼마 증가하는지?)
    2. 포인트 요구치가 상승하고 남은 타일의 위치를 재배치
        (현재 생성된 방해 타일의 근본적인 스타일/현재 포인트 요구치 증가에 대해서는 구현되어 있지 않음)
    3. 목표 포인트 요구치가 크게 늘어남
        (일반 포인트 요구치 증가량과 얼마나 차이를 둘 것 인지)
    4. 포인트 요구치가 상승하고 손패의 카드를 제거함
        (카드를 얼마만큼 제거할지 스테이지 증가에 따라 제거되는 손패 수가 늘어나는지)
    5. 포인트 요구치가 상승하고 방해 블록을 추가함
        (얼마만큼 방해 블록을 늘릴지)

    팀장 의견
    추가할만한 방해블록은 더 이상 없다고 생각함 다른 기믹의 방해 블록은 추후 추가할 시스템인 맵 시스템을 기반으로 설계해도 충분하다고 생각함

    아래 질문의 대한 피디의 의견이 필요함
        1. 현재 방해 블록의 수는 적당한지?
            - 적당하다면 1번째 맵에서 방해 블록은 몇 개가 좋은지?.
            - 방해 블록이 추가된다면 간격이 있는지?
            - 적당하지 않고 특정 패턴 or 계산식이 있으면 좋다고 생각하면 그에따른 응답이 필요.
        2. 방해 블록 스펙에 대한 질문
            - 1,2,4,5번 방해 블록의 포인트 요구치 상승은 얼마면 좋은지?
            - 3번 방해 블록의 포인트 요구치 상승은 기존 요구치 상승 대비 얼마나 늘어나는지?
            - 일반 타일 생성이 왜 패널티인지?
            - 방해 블록이 추가된다면 얼마나 추가되는지?
            - 손패 제거 방식은 강제 삭제 방식인지? 아니면 선택 삭제 방식인지? 제거되는 손패의 수는 얼마인지?
            - 방해 블록을 핸드의 마지막 카드로 건드리면 어떤 처리가 되는지?
            - 동시에 2개 이상으 방해 블록을 타격했을 때 어떤 처리가 우선적으로 진행되는지?
    
    피디 의견
    

도움 블록 (출처: 땅부수기.pptx 4p)
    1. 다음 카드로 타일 파괴시 추가점수 +1 (현재는 +100이라 생각함)
    2. 손패 하나를 묘지로 보내고 현재 카드를 재사용
    3. 다음 상점에 무료 구매권 1장
    4. 카드 1장 드로우
    5. 다음에 사용할 카드 강화

    팀장 의견
    도움 블록 기획 자체가 오래되어 수정이 필요하다고 생각함 팀장이 생각하는 도움 블록은 아래 서술

    1. 다음 카드로 타일 파괴시 추가점수 +1
        현재 파괴된 블록 점수 합 x1.5
        이유: 다음 카드에 대한 메리트가 조금 부족하다고 생각함 현재 이 타일을 포함해 많은 타일을 부수도록 유도한다.
    2. 손패 하나를 묘지로 보내고 현재 카드를 재사용
        현재 카드를 재사용 *카드 사용 UI 작업이 완료된 뒤 추가하는게 좋아보임.
        이유: 손패를 버리고 재사용한다는 것 자체가 메리트라고 보기 어려움.
    3. 다음 상점에 무료 구매권 1장
        10G 제공 or 카드 보상 추가권
        이유: 해당 도움 블록의 경우 게임 초기에 기획되어 있어 게임 씬과 상점이 분리되기 전에 나온 기획임 수정이 필요함
    4. 카드 1장 드로우
        카드 1장 드로우
        이유: 변경이 필요 없어 보임
    5. 다음에 사용할 카드 강화
        다음에 사용할 카드 강화 or 현재 손패에 있는 모든 패 강화
        이유: 사용 카드 강화나 현재 손패에 있는 모든 카드를 강화하는 방식도 괜찮아 보임 단, 손패가 강화 될 경우 덱에 있는 카드가 강화되는지 확인이 필요.

    피디 의견
