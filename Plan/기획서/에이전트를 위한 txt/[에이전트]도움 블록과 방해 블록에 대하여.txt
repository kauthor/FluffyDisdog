# [시스템 참조 문서] 타일 시스템: 혼돈 및 도움 타일

본 문서는 8x8 타일 보드 기반의 덱빌딩 로그라이크 게임에서 사용되는 특수 타일(방해/도움)의 규칙과 로직을 정의한다. AI 에이전트는 이 규칙을 바탕으로 보드 상태를 분석하고 카드의 가치를 판단한다.

---

### 1. 방해 타일 (Obstacle Tiles)
플레이어의 타일 파괴 흐름을 방해하고 디메리트를 주는 요소이다.

| 타일 명칭 | 상태 | 상세 효과 및 제약 사항 |
| :--- | :--- | :--- |
| **혼돈 타일** | **확정** | 일반적인 타일 파괴 액션으로 파괴 불가. 단, 특정 카드의 특수 능력이나 도움 타일의 효과를 통해서만 제거 가능. |
| **맵 디버프 타일** | **검토 중** | 현재 위치한 맵(테마)의 특성에 따라 개별 효과 부여 예정. (명칭 미정) |

---

### 2. 도움 타일 (Help Tiles)
파괴 시 플레이어에게 이로운 효과를 제공하며, 전략적 우선순위가 높은 타일이다.

| ID | 효과 명칭 | 상태 | 상세 로직 (데이터 참조용) |
| :--- | :--- | :--- | :--- |
| H-01 | **파괴 점수 증가** | **확정** | 파괴 시 해당 턴 최종 점수에 0.5배 가산 (최종 점수 = 기본 점수 * 1.5) |
| H-02 | **재사용** | **확정** | 파괴에 사용된 카드를 묘지로 보내지 않고 즉시 손패(Hand)로 되돌림 |
| H-03 | **보물 확인** | **확정** | 맵 내 숨겨진 보물 영역(Hidden Treasure) 중 한 곳의 위치를 강조 표시 |
| H-04 | **카드 드로우** | **확정** | 파괴 즉시 덱(Deck)에서 카드 1장을 드로우 |
| H-05 | **카드 강화** | **검토 중** | 현재 손패에 있는 카드 중 랜덤 또는 선택한 1장의 성능을 강화 |
| H-06 | **도움 타일 증가** | **검토 중** | 보드 내 빈 공간에 랜덤한 도움 타일 2개를 추가 생성 |
| H-07 | **도움 타일 선택** | **검토 중** | 현재 타일을 제외한 도움 타일 목록 중 하나를 선택하여 즉시 발동 |
| H-08 | **방해 타일 제거** | **검토 중** | 보드 위에 존재하는 방해 타일(혼돈 타일 등) 중 1개를 즉시 제거 |
| H-09 | **방해 타일 무력화** | **검토 중** | 1턴 동안 모든 방해 타일의 부정적 효과를 비활성화 |

---

### 3. 시스템 연관 관계 및 판단 기준

#### 3.1. 카드 시스템과의 관계
- **카드 효과 대상:** 일반 타일 외에 '혼돈 타일'을 파괴할 수 있는 키워드가 포함된 카드는 전략적 가치가 높게 평가됨.
- **재사용 로직:** 'H-02(재사용)' 타일 파괴 시, 해당 카드의 소모 비용(Cost) 소모 여부와 관계없이 손패로 복귀시키는 예외 처리가 필요함.

#### 3.2. 점수 계산 로직
- **H-01(점수 증가):** 합산 방식이 아닌 배수 방식(0.5배 추가)으로 적용됨. 여러 개 파괴 시 연산 방식(합연산/곱연산)은 추후 확정 필요.

#### 3.3. 에이전트 행동 우선순위 가이드
1. 보드 내 **혼돈 타일**이 배치를 방해할 경우, 이를 제거할 수 있는 수단(카드 또는 H-08/09 효과)을 최우선 탐색한다.
2. 현재 손패의 카드 효율을 극대화하기 위해 **H-02(재사용)** 및 **H-04(드로우)** 타일을 연계한다.
3. 목표 점수 도달이 임박했을 경우 **H-01(점수 증가)** 타일 파괴를 최우선 목표로 설정한다.

---

### 4. 데이터 테이블 컬럼 정의 (참고)
- `Tile_ID`: 타일 고유 식별자
- `Tile_Type`: OBSTACLE(방해) / HELP(도움)
- `Effect_Value`: 점수 배율, 드로우 수, 생성 수 등 수치 데이터
- `Status`: CONFIRMED(확정) / UNDER_REVIEW(검토 중)
- `Duration`: 효과 지속 시간 (턴 단위, 즉시 발동은 0)
