# [에이전트용 참고 문서] 프로젝트: 유적 파괴(가칭)

## 1. 게임 개요 및 핵심 루프
*   **장르:** 타일 파괴 + 덱빌딩 로그라이크 (엔드리스 지향)
*   **핵심 로직:** 8x8 타일 보드에서 카드를 사용해 타일을 파괴하며, 제한된 손패 내에 목표 점수(할당량)를 달성하여 스테이지를 클리어하는 방식.
*   **현재 상태:** 프로토타입 핵심 규칙 확정 / 그래픽 컨셉 및 세부 밸런스 검토 중.

---

## 2. 시스템 구성 및 상태

### 2.1. 스테이지 및 진행 로직 (확정)
*   **기본 구조:** 8x8 랜덤 타일 배치.
*   **승리 조건:** 목표 파괴 타일 수(Quota) 달성.
*   **패배 조건:** 손패를 모두 소모할 때까지 목표치 미달성, 또는 방해 블록 타격 제한 초과.
*   **보상 체계:**
    *   초과 파괴 타일: 1개당 추가 골드(G) 지급.
    *   퍼펙트 클리어: 방해 타일 제외 모든 타일 파괴 시 보너스 골드 지급.
    *   스테이지 클리어: 보상 카드 선택(3~4개 중 택1), 유물 획득, 골드 획득.
*   **난이도 스케일링 (5스테이지 단위):**
    *   난이도 상승: 방해 블록 증가, 도움 블록 감소, 방해 블록 타격 허용치 감소 등.
    *   보상 상승: 도움 블록 증가, 숨은 보상 증가, 최대 손패 수 확장 등.

### 2.2. 타일 시스템 (확정)
| 타일 종류 | 효과 | 비고 |
| :--- | :--- | :--- |
| **일반 타일** | 파괴 시 1포인트 획득 | 기본 구성 타일 |
| **방해 타일** | 파괴 불가, 타격 시 스택 차감 | 타격 제한 횟수 존재 |
| **도움 타일** | 파괴 시 특수 효과 발동 | 포인트+1, 드로우, 카드 재사용, 강화 등 |
| **숨은 보상** | 파괴 시 무작위 보상 | 목표치 감소, 방해 블록 제거, 즉시 골드 |

### 2.3. 카드 시스템 (확정)
*   **태그 시스템:** 모든 카드는 `#도구`, `#무기`, `#모험가`, `#불`, `#소비` 등의 태그를 가짐 (유물 효과와 연동).
*   **타격 로직:**
    *   **범위:** 1칸 ~ 가로/세로/대각선 전체.
    *   **확률:** 100% 파괴 또는 특정 확률에 의한 파괴 (강화 시 확률 증가).
    *   **특수:** 방해 타격 가능 여부, 사용 시 골드 획득, 사용 시 파괴 확률 영구 증가 등.

## 3. 데이터 테이블 정의 및 참조값

### 3.1. 캐릭터: 모험가 (확정)
*   **시작 자원:** 100G, 덱 8장.
*   **시작 덱:** 삽(2), 부메랑(1), 횃불(1), 마체테(1), 브로드소드(1), 곡괭이(1), 로프(1).
*   **특이사항:** 추가 캐릭터 확장 가능하도록 설계 (검토 중).

### 3.2. 주요 카드 목록 (일부)
| 카드명 | 범위 | 확률/효과 | 태그 |
| :--- | :--- | :--- | :--- |
| **삽** | 1칸 | 100% | #도구, #모험가 |
| **황금 삽** | 1칸 | 100%, 방해타격 가능, 사용 시 골드 | #도구, #황금, #모험가 |
| **횃불** | 1~8칸 | 선택 100%, 주변 50~80% | #도구, #모험가, #불 |
| **로프** | - | 추가 드로우 2~3장 | #모험가, #소비 |
| **기름병** | - | 다음 #불 카드 파괴 확률 100% | #도구, #모험가, #소비 |

### 3.3. 유물 시스템 (검토 중)
*   **숫돌:** #무기 카드로 파괴 시 점수 +1.
*   **윤활유:** #도구 카드로 파괴 시 점수 +1.
*   **잔상 유물:** 파괴된 타일 위치에 잔상을 남겨 추가 점수 획득 기회 제공.

## 4. 상점 및 경제 시스템 (확정)
*   **판매 품목:** 테마 카드 팩(4종), 유물/소모품(3종), 한정 낱장 카드(2종).
*   **리롤:** 골드 소모하여 목록 갱신 (회차마다 비용 상승).
*   **탐사 의뢰 (특수):**
    *   골드를 지불하고 n일(상점 n회 방문) 후 보상 수령.
    *   성공 확률 존재하며, '추가 의뢰'를 통해 성공률 보정 가능.

---

## 5. UI 및 UX 흐름 (확정)
1.  **로비:** 일반 모드 / 계속하기 / 일일 도전 / 도감 / 랭킹.
2.  **준비 단계:** 캐릭터 선택 및 덱 확인, 시작 유물 확인.
3.  **인게임:** 8x8 보드 중심, 좌측 스테이지 정보, 하단 손패 및 덱/묘지 상태 표시.
4.  **결과 창:** 파괴 통계 및 획득 유물 표시, 랭킹 연동.

---

## 6. 에이전트 판단 기준 (Decision Metrics)

1.  **카드 사용 우선순위:**
    *   목표치 달성을 위해 `확률 100%` 카드를 우선하되, `도움 블록` 위치에 광역기(`횃불`, `해일` 등)를 효율적으로 배치할 것.
2.  **경제 운용:**
    *   초반에는 `탐사 의뢰`를 통해 중장기 이득을 취하고, 후반에는 `리롤`을 통해 덱 컨셉(#도구 또는 #무기 시너지)에 맞는 카드/유물 수집.
3.  **리스크 관리:**
    *   방해 블록 타격 제한이 낮아지는 5단위 스테이지에서는 광역기 사용 시 방해 블록 포함 여부를 반드시 체크할 것.
