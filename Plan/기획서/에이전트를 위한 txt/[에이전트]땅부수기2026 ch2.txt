## [에이전트용 참고 문서] 땅부수기 2026 시스템 명세

이 문서는 AI 에이전트가 게임의 핵심 로직, UI 연출, 점수 계산 방식 및 시스템 간 연관 관계를 이해하고 판단 기준으로 활용하기 위해 작성되었습니다.

---

### 1. 게임 기본 정보
*   **장르:** 타일 파괴 + 덱빌딩 로그라이크 (엔드리스 지향)
*   **핵심 루프:** 8x8 보드의 타일을 카드로 파괴하여 스테이지 목표 점수(Target Score) 달성
*   **플레이어블:** 모험가 1종 (확장 예정)

---

### 2. 전투 피드백 및 연출 (Tile Feedback)

에이전트는 타일의 상태 변화에 따라 다음 연출 규칙을 적용합니다.

| 상황 | 연출 동작 | 파티클 효과 | 상태 |
| :--- | :--- | :--- | :--- |
| **타일 타격 (파괴 카드)** | 좌/우 1px 흔들림 | 파편이 아래로 떨어짐 | **확정** |
| **타일 파괴 실패 / 태그만 부여** | 위/아래/좌/우 1px 흔들림 | 작은 먼지만 아래로 떨어짐 | **확정** |
| **상태 표시** | 태그 및 특수 효과 아이콘 표시 | - | **확정** |

---

### 3. 점수 계산 시스템 (Scoring System)

본 게임의 점수 체계는 '태그 점수'와 '타일 점수'의 합산으로 구성됩니다. (발라트로의 족보 시스템과 유사)

#### 3.1 점수 계산 공식
*   **최종 점수 = (타일 파괴 기본 점수 × 배율) + Σ(보유 태그별 점수 × 배율) + 타일 특수 효과 점수**

#### 3.2 점수 정보표 데이터 구조
| 구분 | 항목 | 설명 (판단 기준) | 상태 |
| :--- | :--- | :--- | :--- |
| **태그 점수(좌)** | 기본 점수 x 배율 | 카드가 보유한 태그(예: 도구, 황금)에 따라 합산되는 점수 | **확정** |
| **타일 점수(우)** | 타일 특수 효과 | 타일 자체의 상태나 파괴 시 발생하는 특수 점수 | **확정** |
| **즐겨찾기** | 별(Star) 표시 | 유저가 선택한 태그가 정보표 최상단에 노출 | **확정** |
| **캐릭터 추천** | 자동 즐겨찾기 | 캐릭터별 권장 태그가 기본으로 상단 배치 | **검토 중** |

#### 3.3 타일 상태별 점수 로직
*   **균열:** 기본 파괴 점수와 연동 + 전용 유물 보너스 적용
*   **잔상:** 기본 0점 (배율 0)
*   **연소 파괴:** 기본 점수 0, 배율 1 (0x1)
*   **감전 효과:** (파괴 점수 x 배율 x 0.1) 계산 후 감전 유물 적용
*   **감전 파괴:** (파괴 점수 x 배율) x -1 (감전 상태에서 파괴 시 패널티)
*   **빙결 파괴:** 기본 파괴 점수 연동 + 빙결 유물 적용

### 4. UI 및 인터페이스 시스템

#### 4.1 보상 및 이벤트 팝업 (Folding UI)
*   **접기 기능:** 보상/이벤트 창 상단의 `▽` 버튼 클릭 시 UI가 아래로 내려가며 게임 보드(손패 등) 확인 가능. (게임 진행은 불가능)
*   **카드 선택 로직:**
    *   기존: 카드 클릭 -> 선택 문구 -> 재클릭 (확정)
    *   **변경:** 카드 클릭(체크) -> 하단 `Next` 버튼으로 확정 (**확정**)
    *   **Skip 기능:** 선택한 카드가 없을 경우 `Next` 버튼이 `Skip`으로 변경 (**확정**)
    *   **Select All:** 모든 보상 체크 (자동 다음 화면 이동 없음)

#### 4.2 레이어 및 덱 관리
*   **캐릭터 레이어:** 배경 프리팹 레이어 3번과 4번 사이에 캐릭터 배치 (다리 가림 연출)
*   **덱 리스트:** 세로 스크롤 방식 적용.
*   **화면 전환:** [DECK] / [CARDS] / [TRASH] 텍스트를 통해 현재 보고 있는 리스트 구분.

#### 4.3 상점 및 의뢰 (Request)
*   **의뢰 상태 표시:** 이미지 리소스로 구분 (`RECEIVED` - 수령 완료 / `REQUESTED` - 추가 의뢰됨)
*   **의뢰 진행도:** 상단 체크마크 및 `DAY -n` 텍스트로 추가 의뢰 횟수(최대 3회) 표기.

### 5. 신규 메커니즘: [불안정] 시스템 (Instability)

방해 타일 타격과 관련된 리스크 관리 시스템입니다.

| 항목 | 상세 규칙 | 상태 |
| :--- | :--- | :--- |
| **게이지** | 캐릭터 옆 배치, 최대 10칸 | **확정** |
| **상승 조건** | 방해 타일 타격 시 +1 | **확정** |
| **패배 조건** | 게이지 10 달성 시 게임 오버 | **확정** |
| **등장 시점** | 특정 층에서 유물 '흔들리는 발판' 획득 시 활성화 | **확정** |
| **관련 카드/유물** | 불안정 회복, 방지, 증가 시 보너스 등 [불안정] 태그 보유 | **확정** |

**[검토 중인 유물 효과]**
*   **불안정 룬:** 게이지 하한선 설정 (1, 2, 3... 이하로 내려가지 않음)
*   **무너지는 발판:** 스테이지 종료 시에도 [불안정] 게이지가 초기화되지 않음.

---

### 6. 시스템 간 연관 관계 (Dependency)

1.  **카드 태그 <-> 점수 정보표:** 카드 사용 시 보유 태그에 따라 점수 정보표의 '태그 점수' 섹션 실시간 계산.
2.  **타일 상태 <-> 유물:** 타일 파괴 시 유물 보유 여부에 따라 '타일 점수'에 가산점/감산점 부여 (예: 감전 유물, 빙결 유물).
3.  **UI 접기 <-> 데이터 조회:** 보상 선택 중 보드의 타일 상태나 남은 점수를 확인하여 최적의 보상을 선택하도록 유도.
4.  **불안정 태그 <-> 게임 오버:** 방해 타일을 파괴하는 전략 선택 시 불안정 게이지 수치와 잔여 체력/점수 간의 트레이드오프 발생.
