[크로스파트 협업용] UI 및 연출 기획 요약
출처: 땅부수기2026_ch2.pptx
========================================

■ 타일 타격 연출
- 타격 성공(파괴): 좌우 1px 흔들림 + 파편 파티클 (아래로 떨어짐)
- 타격 실패(파괴 없음) 또는 태그만 부여 시: 상하 1px 추가 흔들림 + 작은 먼지 파티클
- 실제 파티클 모양/개수는 구현 시 변경 가능 (기획안은 참고용)

■ 점수 정보표
- 하단 버튼으로 표기/숨김 가능
- 좌측: 태그 점수 (카드 태그별 파괴 점수 × 배율)
  - 예: 도구+황금 태그 카드 사용 시 → (파괴점수×파괴배율) + (도구점수×도구배율) + (황금점수×황금배율)
- 우측: 타일 점수 (타일 상태별 특수 점수)
  - 균열: 파괴(기본) 점수와 연동
  - 잔상: 기본 0×0점
  - 연소: 기본 0×1점
  - 감전 효과: (파괴점수×파괴배율×0.1)
  - 감전 파괴: (파괴점수×파괴배율) × -1
  - 빙결 파괴: 파괴(기본) 점수와 연동
- 즐겨찾기(별) 기능: 유저가 선택한 항목 상단 고정

■ 보상 획득창 (인게임 팝업)
- 게임 도중 등장 → 접기(▽)/펼치기(△) 버튼 필수
- 접으면 반투명 배경째로 화면 아래로 내려감
- 접힌 상태에서 게임 진행 불가, 손패 확인만 가능
- 하단 클릭 영역: 카드/덱/묘지 상호작용 제외 처리 필요

- 선택 방식: 클릭 1번으로 선택 토글
- Select all 버튼: 전체 체크 (다음 화면으로 넘어가지 않음)
- 선택 수량 표기: (n/m) 형식
- 진행: 우하단 Next→ 버튼 (아무것도 미선택 시 Skip→으로 표기)
- 경고문 표시 기능 필요

■ 이벤트창
- 게임 도중 등장, 접기 버튼 포함
- 이벤트별 이미지 표기
- 내용이 길면 스크롤 가능
- 선택지: 텍스트 최하단 위치, 별도 색상으로 구분
- 조건 미충족 선택지: 줄처리 표시

■ 덱/묘지 보기 UI
- DECK: 덱 목록
- CARDS: 카드 전체 보기
- TRASH: 묘지 보기
- 스크롤 방향: 세로
- 덱↔카드 전체보기 전환 버튼 필요 (← 아이콘 사용)
- 텍스트 아웃라인 폰트: 프리팹화 요청 (특히 상점 영역)

■ 의뢰 UI
- RequestAddArea: 수령 시 RECEIVED / 추가의뢰 시 REQUESTED 이미지 표기
- DAY -N 형식으로 경과 일수 표기
- 추가 의뢰 최대 3회 → 체크마크로 횟수 표기
- RECEIVED 이미지: 불투명 처리 필요 (현재 반투명 버그 있음)

■ 불안정 시스템 UI
- 캐릭터 옆 별도 게이지 표기
- 마우스 오버 시 설명 표시
- 최대치: 10
- 방해 타일 타격 시 +1
- 10 달성 시 게임 오버

■ 레이어 구조 참고사항
- 캐릭터는 배경 3번~4번 레이어 사이에 위치 (다리 가려짐)
- 배경 프리팹을 3번 레이어 앞/뒤 2개로 분리 권장
