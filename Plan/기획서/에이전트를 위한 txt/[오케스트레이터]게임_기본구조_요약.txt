[기획 오케스트레이터용] 게임 기본 구조 요약
출처: 땅부수기.pptx, 땅부수기2026_ch2.pptx
========================================

■ 게임 핵심 구조
- 8x8 타일 보드에서 카드를 사용해 타일을 파괴하여 목표 점수 달성
- 유저는 시작 시 도구 덱(8장)과 100G를 받음
- 판에는 방해 타일(파괴 금지)과 도움 타일이 랜덤 배치됨
- 손패를 모두 소진할 때까지 최대한 타일을 파괴해 할당량 충족

■ 스테이지 진행 흐름
1. 덱에서 손패 드로우 (기본 8장, 유물/캐릭터로 조정 가능)
2. 손패 카드를 사용해 타일 파괴 → 목표 점수 달성
3. 목표 초과 파괴 시 골드로 환산하여 보너스 지급
4. 방해 타일 제외 모든 타일 파괴 시 추가 보너스 골드 지급
5. 스테이지 클리어 → 카드 보상 선택 → 상점 진입

■ 난이도 상승 구조
- 매 스테이지마다 목표 점수 상승
- 5스테이지마다 방해 타일 수 증가 / 도움 타일 수 감소
- 5스테이지 클리어마다 도움 타일 증가 / 손패 확장 등 보상 지급
- 보스 스테이지: 3스테이지마다 등장, 특수 기믹 포함, 추가 보상 제공

■ 게임 종료 구조
- 21스테이지까지 정상 진행
- 22스테이지부터 요구 점수 대폭 상향 → 간접적 게임 종료 유도
- 유저가 원할 경우 무한 모드 계속 가능

■ 상점 구성
- 매 스테이지 클리어 후 상점 진입
- A구역: 낱장 카드 2장 (캐릭터/공용/기능성)
- B구역: 유물, 포션 (랜덤 3개)
- C구역: 팩 4개
- 리롤: 낱장 카드 제외 전체 리롤 가능
- 의뢰 시스템: 상점에서 골드 투자 → 다음 방문 시 보상 수령

■ 카드 보상 시스템
- 스테이지 클리어 시 카드 보상 제공
- 초반: 일반 카드 위주, 후반으로 갈수록 고밸류 카드/카드 제거/강화 카드 확률 증가
- 카드 리롤: 스테이지 시작 전 전체 손패 교체 가능 (횟수 미정)

■ 숨은 보상 (보물 시스템)
- 스테이지당 기본 1개 생성, 타일 아래 숨겨져 있음
- 덮고 있는 타일을 모두 파괴하면 보상 획득
- 등급: 초급(75%) / 중급(20%) / 상급(5%)
- 보상 선택지 5개 제공, 그 중 골드 1개 보장

■ UI/연출 주요 사항 (땅부수기2026_ch2.pptx 기준)
- 타일 타격 시 흔들림 연출 + 파편 파티클
- 점수 정보표: 태그 점수(좌) / 타일 점수(우) 분리 표기
- 보상 획득창: 접기/펼치기 버튼, Select all 버튼, 선택 수량 표기
- 이벤트창: 게임 도중 등장, 접기 버튼 포함
- 불안정 시스템: 방해 타일 타격 시 불안정 게이지 +1, 10 달성 시 게임 오버
